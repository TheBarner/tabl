Hero Realms(2)

Contents:
    unlimited 'health' resource tokens (visible),
    100 'Gold' resource tokens (visible, per-turn),
    a common deck of:
        1x 'Gold',
        2x 'Cycle'.

In a turn:
    'Draw' phase:
        draw 5 cards.
    'Play' phase:
        play any cards,
        buy any cards.

Players start with:
    50 'health' resource tokens,
    0 'Gold' resource tokens,
    a deck of:
        9x 'Gold',
        1x 'Cycle'.

The game ends when:
    a player reaches 10 'Gold' resource tokens or
    a player reaches 0 or less 'Health' resource tokens.

List of cards:
    'Gold' ~gold coin~:
        'Gold' +1.

    'Cycle' ~cycle~:
        draw 1 card.
